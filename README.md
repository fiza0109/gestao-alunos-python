# 🎓 Sistema de Gestão de Alunos (SGA)

Este projeto foi desenvolvido em **Python** como parte da avaliação de programação. O objetivo é criar uma aplicação de consola que permita gerir o registo de alunos de uma instituição, utilizando estruturas de dados dinâmicas.

---

## 📋 FASE 1: Planeamento

* **Nome do Projeto:** SGA - Sistema de Gestão de Alunos
* **Objetivo:** Permitir o armazenamento, consulta e remoção de dados de alunos (Nome, Idade e Curso).
* **Identificação dos Dados:**
    * `nome`: Nome completo do aluno (String)
    * `idade`: Idade do aluno (String/Int)
    * `curso`: Curso em que o aluno está inscrito (String)
* **Estrutura Prevista:**
    * Menu principal com ciclo `while`.
    * Armazenamento em **lista de dicionários**.
    * Funcionalidade extra de estatística (contagem total).

---

## 🛠️ Estrutura do Programa (Menu)

O programa segue a estrutura de menu solicitada:
1.  **Adicionar:** Insere um novo aluno na lista.
2.  **Listar:** Exibe todos os alunos registados.
3.  **Procurar:** Pesquisa um aluno pelo nome.
4.  **Remover:** Elimina um aluno da lista.
5.  **Estatísticas (Fase 4):** Exibe o total de alunos no sistema.
0.  **Sair:** Encerra a execução do programa.

---

## 💻 Tecnologias e Requisitos Técnicos

O código cumpre os seguintes requisitos obrigatórios:
* **Listas e Dicionários:** Para gestão eficiente dos dados.
* **Ciclos e Condições:** Para navegação no menu e lógica de procura.
* **Funções (`def`):** Modularização do código para cada funcionalidade.
* **Boas Práticas:** Variáveis com nomes adequados e código comentado.

1.  Certifique-se de ter o **Python 3.x** instalado.
2.  Faça o download do ficheiro `gestao_alunos.py`.
